Activity 1: UN Sustainability Goals Site

Stage Link Description
Stage 1
The first stage in this process consisted of extracting the photos for the various UN goals and configuring them. The file structure and template were also created. While no visible progress was made in the development of the site, we learned to set up a basic HTML template consisting of body, head, style, and many other tags.
Stage 2
The second stage involved the actual creation of the flip boxes. Using W3 schools, we utilized code in order to teach us how to implement flip boxes into our page.
Stage 3
The third stage finally displayed all 17 of the goals in a single column. This stage also consisted of changing the colour of the back of the flip box in order to match the colour on the front.
Stage 4
The fourth stage allowed us to format the flip boxes in rows and columns rather than having just one column. In addition, as the page becomes more narrow, the flip boxes would proceed to the next line.
Stage 5
The fifth stage involved creating a header for the site using the Sustainable Development Goals logo that was found. Upon compressing the page, the image will collapse and become the square version of the logo instead. This knowledge can be applied in all forms of pages in order to collapse the page.
Stage 6
The sixth stage taught us how to make a fixed header on a website. This can be used in very important aspects of different sites such as maintaining a menu bar at the top of a page.
Stage 7
In the seventh stage we learned how to display the questions/information into the back of the flip box.

Activity 2: JSON Data - HIV Infeaction Rate By Country

Stage Link Short Description
Stage 1
In this stage we will be checking to ensure that the site is properly loading JQUERY
Stage 2
The second stage involved accessing the JSON file that consisted of data as well as storing it.
